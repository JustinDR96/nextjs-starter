# 📁 constants

Ce dossier contient toutes les **constantes globales** du projet.

Cela inclut :

- les routes (`routes.ts`)
- les messages, labels ou couleurs constantes
- les rôles utilisateurs, etc.

---

## Exemple :

- `routes.ts` : chemins centralisés pour la navigation
- `userRoles.ts` : valeurs fixes de rôles d’utilisateur
